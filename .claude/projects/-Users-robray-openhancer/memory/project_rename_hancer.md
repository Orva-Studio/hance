Product renamed from "openhancer" to "hancer" on 2026-04-01. The "open" prefix signaled free/open-source and undermined commercial positioning.

**Why:** Preparing to sell the product — "open" in the name conflicted with a paid licensing model (FSL-1.1-Apache-2.0).

**How to apply:** Use "hancer" everywhere in new code. The GitHub repo may still be named "openhancer" — the directory name hasn't changed. Old docs/plans still reference "openhancer" and were intentionally left as historical records.
